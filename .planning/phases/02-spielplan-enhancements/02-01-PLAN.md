**Function signature to implement:**
```js
function renderPastSection(pastSorted, teamId, visibleCount) { ... }
```
- `pastSorted`: array of `{id, gd, games}` entries, sorted newest-first (same structure as cache.past)
- `teamId`: number
- `visibleCount`: how many entries to show (not `cfg.past` — this changes as user clicks)
- Returns: HTML string showing visibleCount cards + "Weitere laden (N)" button if more remain

**Button spec:**
- Class: `load-more`
- `data-team` attribute: teamId (for click delegation lookup)
- `data-visible` attribute: current visibleCount (click handler reads this to increment)
- Label: `Weitere laden (N)` where N = remaining count

**Constraining decision (01-02):** Both renderers are top-level functions accessible from VM context.
