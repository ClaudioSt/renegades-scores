<feature>
  <name>computeStandings</name>
  <files>standings.js, tests/standings.test.js</files>
  <behavior>
    computeStandings(leagueConfig, gamedays) returns standings per league+season:

    - Only gamedays whose `id` appears in `cfg.gameday_ids` for a league+season are counted
    - Only games with `final_score != null` count toward standings (skip in-progress/scheduled)
    - Each game has `results: [{ team_id, team_name, pa, isHome }]` — skip games with fewer than 2 results
    - For each counted game: lower `pa` wins; equal `pa` is a draw
    - EP (points scored by team A) = opponent's `pa` in that game
    - GP (points conceded by team A) = team A's own `pa` in that game
    - SQ = (2 × S + 1 × U) / (2 × Sp) — result is a float rounded to 4 decimal places
    - Rows sorted: SQ desc, then PD desc, then EP desc (all three deterministic)
    - `promotion_restricted` array is passed through from `cfg.promotion_restricted` unchanged
    - An empty `gameday_ids` array → `rows: []`
    - Games where a team has no matching entry in results → that team simply doesn't get a row

    Cases:
    - teamA.pa=7, teamB.pa=14 → A wins (A.S=1, A.EP=14, A.GP=7), B loses (B.N=1, B.EP=7, B.GP=14)
    - teamA.pa=7, teamB.pa=7 → draw (both U=1, both EP=7, GP=7)
    - game has final_score=null → not counted
    - gameday.id not in any gameday_ids → not counted
    - empty leagueConfig → returns {}
  </behavior>
  <implementation>
    Create `standings.js` as CommonJS module.
    Iterate leagueConfig entries → seasons → cfg.
    Build a Set of gameday IDs for each league+season.
    Filter withGames to matching gameday IDs.
    For each matching gameday, iterate games where `final_score != null` and `results.length >= 2`.
    Use a `teamStats` map keyed by `team_id`.
    Determine winner/loser/draw by comparing `pa` values of the two results.
    Build rows from teamStats, compute SQ = parseFloat(((2*S + U) / (2*Sp)).toFixed(4)).
    Sort rows: SQ desc → PD desc → EP desc.
    Return structured object.
  </implementation>
</feature>
