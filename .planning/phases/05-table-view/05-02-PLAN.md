Where `contentHtml` is the existing HTML built by iterating `entriesForYear`. Introduce a helper:

```javascript
function _renderStandingsForYear(allEntries, year, teams) {
  var entries = allEntries.filter(function(e) { return e.year === year; });
  var html = '';
  entries.forEach(function(e) {
    html += '<div class="standings-league-title">' + escapeHtml(e.entry.name) + '</div>';
    html += renderStandingsTable(e.entry, teams);
  });
  return html || '<div class="standings-empty">Keine Tabellendaten für ' + year + '.</div>';
}
```

```javascript
// Module-level state for year switching (set by renderTableView)
var _standingsAllEntries = [];

```

Inside `renderTableView`, after building the initial HTML and calling `root.innerHTML = ...`, set these vars:

```javascript
_standingsAllEntries = allEntries;

```

Then add a click listener on `root` for year-tab clicks (use event delegation, same pattern as `.load-more` in `renderSpielplan`):

```javascript
root.addEventListener('click', function(e) {
  var btn = e.target.closest('.year-tab');
  if (!btn || !_standingsRoot) return;
  var year = parseInt(btn.getAttribute('data-year'), 10);

  // Update active tab styling
  _standingsRoot.querySelectorAll('.year-tab').forEach(function(b) {
    b.classList.toggle('year-tab-active', parseInt(b.getAttribute('data-year'), 10) === year);
  });

  // Re-render content for selected year (tabs wrapper stays, only content swapped)
  var contentEl = _standingsRoot.querySelector('.standings-year-content');
  if (contentEl) {
    contentEl.innerHTML = _renderStandingsForYear(_standingsAllEntries, year, _standingsTeams);
  }
});
```

To make the content-swap work, wrap the initial `contentHtml` in a `<div class="standings-year-content">` so the click handler can target it:

```javascript
root.innerHTML = '<div class="standings-wrapper">'
  + tabsHtml

```

Add the `.standings-year-content` wrapper CSS (no special styling needed, just a container):
