- Signature: `renderStandingsTable(entry, teamIds)` where:
  - `entry` = `{ name, rows: [{team_id, team_name, Sp, S, U, N, EP, GP, PD, SQ}], promotion_restricted: [] }`
  - `teamIds` = array of integer team IDs to highlight
- Returns HTML string

**Behaviors to test (write failing tests for ALL before implementing any):**
1. Empty rows → returns div with "Keine Tabellendaten" text
2. Non-empty rows → returns `<table>` element
3. Column headers present: Mannschaft, Sp, S, U, N, EP, GP, PD, SQ
4. Row count matches entry.rows.length
5. team_name appears in rendered HTML
6. Team ID in teamIds → row has class `standings-row-highlight`
7. Team ID NOT in teamIds → row does NOT have class `standings-row-highlight`
8. Team in promotion_restricted → row has class `standings-row-restricted`
9. Team NOT in promotion_restricted → row does NOT have class `standings-row-restricted`
10. SQ value displayed with 4 decimal places (e.g. 0.9167 → "0.9167")
11. XSS: team_name containing `<script>alert(1)</script>` is escaped — raw tag not in output
12. XSS: team_name containing `"><svg onload=alert(1)>` is escaped
